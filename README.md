# Calculator
Calculator in progress.  
Link to calculator: https://accountexeregister.github.io/Calculator/

Keyboard inputs (left): Input to calculator (right)  
0 - 9 => 0 - 9  
\+ => +  
\- => -  
x => *  
/ => /  
. => .  
= => =  
Alt => (+ / -) (inverse sign)  
< => %  
Backspace / <- => UNDO  
Del => RESET NUM  
Esc => CLEAR
